How I Made My Presentation

A.K.A. Why you should use reveal.js

Easy Setup

  1. Clone the reveal.js repository
  2. Open index.html in a web browser to view your presentation

Full Setup

  1. Install Node.js version 10.0 or greater
  2. Clone the reveal.js repository
  3. Move to the directory and install via npm install
  4. Serve the presentation via npm run
  5. Open http://localhost:8000 to view your presentation

How do you edit the presentation?

The Basics

  • Entire presentation can be in index.html
  • The starting index.html has reveal.js baked in
  • Each slide is contained in a <section> tag
  • Write your HTML and it will render!
  • Many examples of reveal.js specific functionality online
